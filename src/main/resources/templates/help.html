Назначение системы

Данная система предназначена для моделирования работы железнодорожной станции.

Главное меню

В главном меню расположены пункты меню для работы с системой в соответствии с её реализуемыми функциями: конструктор топологии ЖД станции, менеджер расписаний, моделирование и визуализация и справочная информация о программе.

В верхней части главного меню отображается активный на данный момент пользователь и кнопка «Выйти», позволяющая ему выйти из системы.

Для начала работы с системой выберите необходимую для вас функцию.

Конструктор топологии ЖД станции

При выборе в меню пункта «Конструктор топологии ЖД» отобразится список станций с которыми можно работать.

Пункт «Создать новую...» позволяет создать новую топологию.

После выбора пункта «Создать новую...», в появившемся модальном окне необходимо ввести её имя, и выбрать город в котором она расположена.

После введения всех данных необходимо нажать на кнопку «Создать» после чего можно приступить к работе с ее топологией.

Для выхода из окна добавления станции необходимо нажать на кнопку «Отмена» после чего окно закроется.

При переходе на главную страницу конструктора топологий, на ней отображается топология станции

В боковой части конструктора расположены формы для добавления с заданными параметрами элементов топологии

Для добавления определенного элемента топологии необходимо задать параметры нужного элемента и нажать на кнопку «Добавить».

Для изменения параметров элемента на топологии, необходимо нажать на него мышью, после чего отобразится форма для изменения его параметров и полного удаления его из топологии.

Для изменения элемента топологии необходимо задать нужные значения в поля и нажать кнопку «Изменить».

Для удаления элемента топологии необходимо нажать кнопку «Удалить».

Для возврата в главное меню необходимо нажать на кнопку в левой верхней части окна. Все изменения будут сохранены автоматически.

Менеджер расписаний

При выборе в меню пункта «Менеджер распианий» отобразится список станций с распианием которых можно работать.

При выборе станции отобразится таблица расписания данной станции.

Для добавления в таблицу расписания поезда, неодходимо нажать кнопку «Добавить»

Для изменения в таблице расписания поезда, неодходимо нажать на строку нужного поезда

При нажатии кнопки «Добавить» отобразится форма для введения параметров нового поезда

После введения данных о новом поезде необходимо нажать кнопку «Создать».

Для выхода из формы создания поезда необходимо нажать кнопку «Отмена».

При нажатии на строку поезда в таблице, откроется форма для изменения его параметров

После введения новых данных о поезде необходимо нажать кнопку «Изменить».

Для удаления поезда необходимо нажать кнопку «Удалить».

Для возврата в главное меню необходимо нажать на кнопку в левой верхней части окна. Все изменения будут сохранены автоматически.

Моделирование и визуализация

При выборе в меню пункта «Моделирование и визуализация» отображается список станций для моделирования и последующей визаулизации работы.

При переходе на страницу визуализации, на ней отображается визуализация работы станции.

На панеле управления визуализацией присутствуют кнопки для выбора режима визуализации.

Для приостановки процесса визуализации необходимо нажать кнопку «Пауза».

Для старта и возобновления процесса визуализации необходимо нажать кнопку «Старт».

Для остановки процесса визуализации необходимо нажать кнопку «Стоп».

На панеле управления визуализацией отображается текущее модельное времени.

На панеле управления визуализацией присутствует слайдер для управления течения скорости модельного времени.

Для возврата в главное меню необходимо нажать на кнопку в левой верхней части окна. Все изменения будут сохранены автоматически.
